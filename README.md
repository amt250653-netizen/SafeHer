## 🛡️ SafeHer – Women's Safety Application

SafeHer is an Android-based women's safety application designed to provide immediate assistance during emergencies. The application enables users to send SOS alerts, share their live location, notify emergency contacts, and activate multiple safety features with a single action.

---

## 📱 Features

- 🆘 Hold-to-Activate SOS
- 📳 Shake Detection for Emergency Activation
- 📍 Real-Time Live Location Sharing
- 📶 Offline SMS Alert when Internet is unavailable
- 📸 Silent Evidence Capture
- 📞 Fake Incoming Call
- 👨‍👩‍👧 Emergency Contacts Management
- 📜 SOS History
- ✅ "I'm Safe" Button

---

## 🚀 How to Run

1. Clone the repository.
2. Open the project in Android Studio.
3. Sync Gradle.
4. Add your Firebase configuration (`google-services.json`) if needed.
5. Build and run the application on an emulator or Android device.

---

## 🔒 Purpose

The objective of SafeHer is to improve personal safety by enabling users to quickly notify trusted contacts, share their location, and activate emergency features during critical situations.
